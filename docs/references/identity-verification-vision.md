# 視覺身分驗證模式 (Identity Verification via Vision)

在自動化 LINE 或其他通訊軟體時，僅靠文字搜尋可能誤入「私人群組」而非「官方帳號」。

## 模式流程
1. **執行搜尋**：輸入關鍵字並進入第一個結果。
2. **截圖驗證**：在發送任何機密資訊前，截取對話視窗。
3. **視覺分析**：使用 `vision_analyze` 確認以下特徵：
   - **標題欄**：是否顯示為官方帳號（帶有盾牌標誌）？還是顯示為成員數 (N) 的群組？
   - **頭像**：是否為店家商標？
   - **對話歷史**：是否有廣告、自動回覆或官方導覽？
4. **決策**：若確認為私人群組且任務為官方業務，應立即停止並嘗試搜尋「官方帳號 (Official Accounts)」分頁。

## 案例紀錄 (2026-05-10)
- **場景**：預約「小明火鍋店」。
- **錯誤**：點擊了名為「小明」的聯絡人，結果進入了一個 4 人朋友群組。
- **修正**：透過截圖發現群組標題無官方標誌，且對話內容包含朋友間的「拼車」閒聊，成功避免在錯誤群組中進行訂位溝通。
